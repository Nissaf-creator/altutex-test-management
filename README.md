# Altutex Test Management

A web-based dye test management system for textile articles. This project allows tracking of dye tests for various materials, storing test results, and calculating success rates.

---

## Features

- Manage dye tests for multiple articles and clients.
- Store test data in JSON format.
- Calculate success rates automatically based on the number of tests required.
- Display data using charts for easier analysis.
- Search and filter test results.

---

## Technologies Used

- **Backend:** PHP 8+  
- **Database:** MySQL 5.7+  
- **Frontend:** HTML, CSS, JavaScript, jQuery, Bootstrap, Datatables, Select2, Chart.js  
- **Tools:** XAMPP for local development, Git for version control  

---

## Requirements

- XAMPP (Apache + MySQL) installed.
- Web browser (Chrome, Firefox, Edge).
- PHP enabled in XAMPP.

---

## Installation / Setup

1. Clone the repository:

```bash
git clone https://github.com/Nissaf-creator/altutex-test-management.git 
```


2. Place the project in your XAMPP htdocs folder:

C:\xampp\htdocs\altutex-test-management

3. Start Apache and MySQL in XAMPP.

4. Open the project in your browser:

http://localhost/altutex-test-management

> The database and tables are automatically created on first load with sample test data.
